🎁 IMAGES DIRECTORY (public/assets/images/)
=======================================

Put all your scrapbook photos and memory images in this folder!

Expected File Names:
- `day-01.jpg`
- `day-02.jpg`
- `day-03.jpg`
- `day-04.jpg`
- `day-05.jpg`
- `day-06.jpg`
- `placeholder.jpg` (used as the frame background for locked/coming-soon days)
- `birthday.jpg` (your absolute favorite picture, revealed on July 22nd)

Important Tips:
1. Crop or edit your images to be square (1:1) or standard portrait aspect ratio (like 4:5) so they sit perfectly inside the white Polaroid photo border.
2. If you don't have a picture for a day yet, the website automatically shows a beautiful warm heart-and-camera placeholder instead of a broken link.
3. When you add a new photo (e.g. day-07.jpg), update the "image" key in /public/assets/data/days.json from "placeholder.jpg" to "day-07.jpg" to display it!
